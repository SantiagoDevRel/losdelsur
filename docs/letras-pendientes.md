# Letras pendientes - La Banda Los Del Sur

54 cánticos sin letra verificada. Copiá el prompt de abajo por cada canción y pegalo en Claude browser.

## Prompt base (editá el título/CD/año por cada canción)

```
Hola Claude, necesito la letra exacta del cántico de la barra Los Del Sur
titulado "{TITULO}" del CD {CD_NUMERO} (año {AÑO}).

Devolveme SOLO la letra, línea por línea como se canta. Sin encabezados,
sin comentarios, sin acordes, sin marcas de coro. Si hay coros que se
repiten, escribilos completos cada vez (no uses "(bis)" ni abreviaturas).

Las letras son informales, en voseo paisa. Fuentes posibles:
canticosdelsur.com, letras.com, YouTube, foros de hinchas de Nacional.
Si no la encontrás, decilo - no inventes.
```
